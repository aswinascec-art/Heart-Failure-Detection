# Heart Failure Detection

## Project Description
Machine learning model to predict heart failure using patient health data.

## Dataset Features
- Age, Sex, Chest Pain Type
- Resting Blood Pressure, Cholesterol
- Fasting Blood Sugar, Resting ECG
- Max Heart Rate, Exercise Angina
- Oldpeak, ST Slope

## How to Run
1. Open `Techmaggii.ipynb` in Google Colab
2. Run all cells

## Results
Predicts whether a patient has heart disease (1) or not (0).
